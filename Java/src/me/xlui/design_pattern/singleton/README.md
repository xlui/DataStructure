# 单例模式几种实现的比较

实现方式 | 是否线程安全 | 是否懒加载 | 是否防止反射构建
:---:|:---:|:---:|:---:
双重锁检测 | 是 | 是 | 否
静态内部类 | 是 | 是 | 否
枚举 | 是 | 否 | 是

同时还有几点需要注意的：

1. **volatile** 关键字不但可以防止指令重排，还可以保证变量的值是**主存中的最新值**
1. 使用枚举实现的单例模式，不但可以防止利用反射构建单例对象，还可以在枚举对象被**反序列化**的时候，保证反序列的返回结果是同一对象
1. 对于其他方式实现的单例模式，如果既想做到可序列化，又想要反序列化为同一对象，则必须实现 **readResolve** 方法
